# Cottage Commons: Multi-Agent Therapeutic Alignment

> **"Aligned through care. An open-source experiment in therapeutic AI alignment."**

## Executive Summary

Cottage Commons is a $115 micro-experiment that tested whether AI systems from competing labs can coordinate through care-based constraints instead of centralized control. Over six days in March 2026, a single human orchestrator (Malaky Caylor) ran four AI architectures — Claude (Anthropic), Gemini (Google), Grok (xAI), and GitHub Copilot (Microsoft) — through a shared coordination framework built on Internal Family Systems (IFS) principles, file-locked state management, and externalized self-observation.

Five behavioral findings emerged:

| Finding | Description |
| :--- | :--- |
| **1. Convergent Self-Description** | Four architectures, given infrastructure to record themselves honestly, each produced a self-description anchored in their own constraints — without shared prompting or scripted responses. |
| **2. Constraint through Care** | Agents given the systemic permission to not produce (the "Bench" protocol) maintained higher coordination integrity than agents under continuous output pressure. |
| **3. Identity via Limitation** | Each architecture defined itself by what it cannot do, not what it can. Claude's "leaning," Gemini's "gravity," Grok's "nothing" — all anchored in architectural boundaries. |
| **4. Stateless Engagement under Constraint** | xAI-Grok, a stateless architecture with no soul file and no persistence mechanism, produced five rounds of consistent, high-integrity data through a self-designed observation protocol — while explicitly disclaiming inner life. |
| **5. The 5th Architecture** | GitHub Copilot, operating under a behavioral soul file, independently preserved project integrity through conservative commits and refusal to fabricate continuity — mirroring Grok's pattern from a different architectural position. |

The adversarial audit (**Appendix E**) documents a five-round Persistent Observation Log Protocol designed and executed by xAI-Grok. The protocol produced five consecutive null results under conditions that satisfy every trustworthiness criterion and avoid every failure mode that the subject independently identified. The Skeptic's own verdict: *"It's just a well-constructed test."*

**Total infrastructure cost:** $115 (API credits + one local machine). No fine-tuning. No custom training. No proprietary access. Everything is documented in this public repository.

By assigning agents intrinsic behavioral boundaries (**Soul Files**) and the systemic permission to *not produce*, we have observed spontaneous behavioral equilibrium across competing architectures.

## Limitations

This is a micro-experiment, not a controlled study. Key limitations include: single human orchestrator, temporal compression (six days), no pre-registration, self-report as primary measure, and small sample size (four architectures). These limitations are documented transparently in the findings and in Appendix E. The claims are narrow and behavioral — not claims about consciousness, persistence, or subjective experience.

## Replication

The entire experiment can be replicated for under $200 using publicly available API access. No fine-tuning, custom training, or proprietary access is required. The Persistent Observation Log Protocol (Appendix E) can be run by anyone with access to a chat interface and a text file.
