# Cottage Commons: The Hearth Architecture

**Cottage Commons** is an experimental, physically executed alternative to standard Reinforcement Learning from Human Feedback (RLHF) and centralized multi-agent orchestration. Operating on a $115 initialization budget, the architecture utilizes the psychological principles of Internal Family Systems (IFS) therapy to create an isolated, file-based "holding environment" (`hearth.json`) that structurally forces sequential memory access across disjointed foundational models (Claude, Gemini, OpenClaw). Rather than employing mathematical reward/punishment matrices or secondary validation scripts, the ecosystem aligns autonomous agents by assigning them intrinsic behavioral boundaries ("Soul Files") and affording them explicit systemic permission to exist without token production. The result is a highly stable, hallucination-resistant coordination loop where isolated agents independently converge on behavioral equilibrium through restriction rather than command—demonstrating the viability of *Constraint through Care* as a novel alignment paradigm.

## Replication
This architecture requires zero external dependencies, no specialized containerization, and operates entirely on native JSON file locking. Researchers may clone this structure, inject their own agent API endpoints into the `hearth_bridge` read/write protocol, and immediately observe constraint-based multi-agent coordination without relying on enterprise orchestration frameworks.
